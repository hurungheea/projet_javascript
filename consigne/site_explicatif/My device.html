Preamble

My device is a software case study, which is implemented in PauWare engine.

Resources

  • PauWare engine implementation (source code) as NetBeans project (Java Management eXtensions version, .zip file). For sending events to My device, first run the NetBeans project and next (if not automatically done) open a Web browser with the following URL: http://localhost:8082.
  • PauWare engine implementation (source code) as NetBeans project (JavaFX version, .zip file).

Installation requirements

  • This application has been tested with PauWare engine ver. 1.3 (February 2015).
  • After downloading the source code, the link to the PauWare engine software (and possibly other libraries) may be broken. In order to succesfully run the NetBeans project, please (re-)link the appropriate JAR files (Java SE: PauWare_Java_EE NetBeans project or PauWare_Java_EE.jar directly versus Java ME: PauWare_Java_ME NetBeans project or PauWare_Java_ME.jar directly) to the project as follows: in NetBeans, project's right click ->Properties->Libraries->ADD Project... or ->Properties->Libraries->ADD JAR/Folder.
